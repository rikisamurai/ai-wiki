# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

中文 Obsidian 知识库，主题涵盖 AI 编程工具、AIGC、Obsidian 使用技巧等。

## 项目结构

- `😈Magic/` — 工具与技术笔记（AI 编码、Obsidian、网络等）
- `🚀AIGC/` — AIGC 相关内容（AI 编码工具如 Claude Code、Superpowers 等）
- `.agents/skills/` — Claude Code skills（obsidian-markdown、obsidian-cli、json-canvas 等）

## 笔记规范

- 笔记内容尽量使用**中文**撰写，但英文专有名词不要使用中文（比如写“Agent”，而不是写“代理”)
- 使用 Obsidian Flavored Markdown（wikilinks、callouts、frontmatter 等）
- 内部链接使用 `[[wikilink]]`，外部链接使用标准 Markdown 语法
- 新建笔记必须包含 YAML frontmatter，格式参考：
  ```yaml
  ---
  title: 笔记标题
  tags:
    - tag1
    - tag2
  date: YYYY-MM-DD
  ---
  ```
- 善用 Obsidian callouts（`> [!type]`）组织内容
